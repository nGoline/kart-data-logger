# SMD PCB Version

This folder contains the surface-mount board design files for the kart telemetry project.

Available files:

* `kart_smd.kicad_pcb` — SMD PCB layout.
* `kart_smd.kicad_sch` — schematic for the SMD board.
* `.history/` — KiCad project history and backups.

Use this variant when you want a compact, production-ready board with surface-mounted components.
