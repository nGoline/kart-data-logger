# Through-Hole Perfboard Version

This folder is intended for the through-hole perfboard version of the kart PCB.

It should contain layout, wiring, and assembly information for a perfboard-based build alternative.

## Purpose

The perfboard variant is useful when a full custom PCB is not available or when hand assembly is preferred.

## Status

The actual through-hole layout files are not included yet. Add the perfboard schematic, board layout, and component placement documentation here when they are available.
