# PCB Documentation

This folder holds PCB design documentation for the kart telemetry system.

There are two PCB variants documented here:

* `hardware/pcb/SMD/README.md` — the surface-mount device (SMD) PCB version.
* `hardware/pcb/Perfboard/README.md` — the through-hole perfboard version.

Use this root README as the entry point for the PCB designs and follow the subdirectory README files for variant-specific details.
